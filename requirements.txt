Environment Tips
Avoid using Python 3.12 (some g4f builds break on it).
Run in a virtual environment to isolate packages:

python -m venv venv
venv\Scripts\activate   # Windows
source venv/bin/activate  # macOS/Linux

Always test your build by reinstalling on a fresh clone.
